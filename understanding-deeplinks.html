Understanding deeplinks

When loading links in a page there are a number of different ways that these can trigger a deeplink into a given app. Some of these can give a sub-optimal user experience on different device / browser combinations.

These are the links triggered from tapping the booking.com ad generated for the query "hotels st pancras".

  1. [google.com] googleadservices.com redirect
  2. [booking.com] HTTP target, redirect to HTTPS
  3. [booking.com] HTTPS target
